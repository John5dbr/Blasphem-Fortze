Sobre o Projeto

Blasphem Fortze é um jogo de piano interativo rodando direto no navegador, sem precisar instalar nada. A ideia é simples: escolha uma música, acompanhe as teclas que aparecem na tela e pressione as teclas certas no tempo certo para pontuar.
O projeto foi construído inteiramente com HTML, CSS e JavaScript puro, sem frameworks.

Toda a lógica do jogo, desde a seleção de músicas até o sistema de pontuação em tempo real, é coordenada por scripts que se comunicam via eventos customizados, mantendo o código organizado e desacoplado.

Interface do Jogo

deve-se encostar o mouse no lado
direito da tela.
